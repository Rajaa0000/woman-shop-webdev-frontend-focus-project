# Rajaa Shop

Rajaa Shop is an online women’s store created with modern web technologies. It allows users to browse and purchase products across different categories, including shoes, accessories, and clothes. The website is designed to provide a smooth shopping experience with dynamic content and an attractive interface.

## About

Rajaa Shop is a responsive e-commerce website for women’s fashion. It offers:
- Categories for Shoes, Accessories, and Clothes
- Individual product pages with details and purchase forms
- A landing page showing a glance at all product categories and the latest additions
- Additional pages like Other Stores, Get in Touch, Privacy, and Terms

---

## Features

- Responsive design for mobile and desktop
- Dynamic routing for product and category pages
- Product listing by category
- New products section highlighting latest additions
- Detailed product pages with purchase forms
- Smooth animations using Framer Motion
- Clean and modern UI

---

## Technologies

- **Next.js** – React framework for server-side rendering and static site generation
- **React.js** – Frontend library for building UI components
- **Sanity.io** – Headless CMS for managing product data
- **Tailwind CSS** – Utility-first CSS framework for styling
- **Framer Motion** – Animations for interactive UI

## Installation

1. Clone the repository:  git clone https://github.com/username/rajaa-shop.git
2. Navigate into the project folder: cd rajaa-shop
3. Install dependencies: npm install
4. Start the development server:npm run dev
5. Open in your browser: http://localhost:3000

--- 

## Usage

Browse categories: Shoes, Accessories, Clothes
Click a product to view details
Fill out the purchase form to order a product
Explore new products section on each category page
Navigate additional pages: Other Stores, Get in Touch, Privacy, Terms
